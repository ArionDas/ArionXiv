# Environment Variables Quick Reference

## REQUIRED (Must Set)

```bash
MONGODB_URI=mongodb+srv://<username>:<password>@<cluster>.mongodb.net/?retryWrites=true&w=majority
JWT_SECRET_KEY=<generate-strong-secret>
```

**At least ONE LLM provider:**
```bash
OPENROUTER_API_KEY=sk-or-v1-...
# OR
GROQ_API_KEY=gsk_...
```

---

## RECOMMENDED (Better Performance)

```bash
# Both LLM providers for redundancy
OPENROUTER_API_KEY=<your-openrouter-key>
GROQ_API_KEY=<your-groq-key>

# Gemini for FREE embeddings
GEMINI_API_KEY=<your-gemini-key>

# Database name
DATABASE_NAME=arionxiv

# Model preferences
DEFAULT_ANALYSIS_MODEL=llama-3.3-70b-versatile
RAG_LLM_PROVIDER=openrouter
OPENROUTER_MODEL=meta-llama/llama-3.3-70b-instruct:free
```

---

## OPTIONAL (Has Sensible Defaults)

```bash
# JWT Configuration
JWT_EXPIRE_HOURS=24

# ArXiv Settings
ARXIV_MAX_RESULTS=50
ARXIV_SEARCH_DAYS=7

# Daily Analysis Cron (UTC)
DAILY_CRON_HOUR=6
DAILY_CRON_MINUTE=0
TIMEZONE=UTC

# Security
PASSWORD_SALT_LENGTH=32
PBKDF2_ITERATIONS=100000

# MongoDB Connection Tuning
MONGODB_CONNECT_TIMEOUT=30000
MONGODB_SERVER_SELECTION_TIMEOUT=30000
MONGODB_SOCKET_TIMEOUT=30000
MONGODB_MAX_POOL_SIZE=10
MONGODB_RETRY_WRITES=true
MONGODB_RETRY_READS=true
```

---

## Security Notes

1. **JWT_SECRET_KEY**: Let Render auto-generate this (click "Generate")
2. **API Keys**: Never commit these to Git
3. **MongoDB URI**: Create a separate DB user for production
4. **.env file**: Already in `.gitignore` - DO NOT COMMIT

---

## Verify After Setting

After setting env vars on deployment platform, check logs for:
```
MONGODB_URI (Database connection)
JWT_SECRET_KEY (Authentication)
OPENROUTER_API_KEY or GROQ_API_KEY (LLM)
```

If any are missing, the health check will work but features will fail.
